# Cherri shortcuts

Estos archivos son fuentes para Cherri Playground:

- `Gym-Agua.cherri`
- `Gym-Peso.cherri`
- `Gym-Entreno.cherri`
- `Gym-Modo.cherri`

Abre cada archivo, copia el contenido en Cherri Playground y exporta el shortcut con el mismo nombre:

- `Gym Agua`
- `Gym Peso`
- `Gym Entreno`
- `Gym Modo`

La PWA llama esos atajos con `shortcuts://run-shortcut`.

Nota: una PWA no puede crear widgets nativos de iPhone ni activar un modo de concentracion por si sola. Para eso se usa Atajos/iOS. El dashboard deja los botones preparados y los archivos Cherri para exportarlos desde https://playground.cherrilang.org/.
